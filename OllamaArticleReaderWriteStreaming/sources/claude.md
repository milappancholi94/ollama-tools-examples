# Raw notes — Claude (Anthropic's AI assistant)

- Claude is a family of large language models made by Anthropic
- newest models: Claude Fable 5 (most capable), Opus 4.8, Sonnet 5, Haiku 4.5
- good at: coding, long agentic tasks, writing, analysis, tool use
- large context window (up to 1M tokens on recent models)
- adaptive thinking: the model decides how much to reason per task
- tool use / function calling: Claude can call your tools to take actions
- available via API, claude.ai web app, Claude Code CLI, desktop + IDE extensions
- safety focus: Anthropic emphasizes helpful, honest, harmless behavior
- Claude Code = agentic coding tool that runs in the terminal
